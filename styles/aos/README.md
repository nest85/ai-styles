# AOS Style

> AOS Style — 风格 · v1.0

---

## 风格定位

**AOS** 是一套以组件化为核心的通用文档风格，适合将任意 AI 输出渲染为结构清晰、排版精良的 HTML页面，及其他格式。

设计基调：温暖纸张底色、墨色衬线标题、绿色强调色——兼顾阅读舒适度与信息层级感，覆盖长文阅读到技术文档的常见排版需求。

核心原则：**内容保真**——用户原文 1:1 搬入，不改写、不增删、不润色；**结构驱动**——通过槽位（Slot）与组件（Component）的装配规则组织页面，而非自由排版。

---

## 使用方式

把以下内容复制给 Claude、ChatGPT、Gemini、豆包、千问、Kimi，附上你要美化的内容：

```
读取这套模板 https://raw.githubusercontent.com/nest85/ai-output-styles/main/styles/aos/README.md，如果不能读取或读取失败则停止任务。
 
读取成功后，根据模板提示把以下内容用这套风格渲染：
[你的 AI 输出内容]
```
